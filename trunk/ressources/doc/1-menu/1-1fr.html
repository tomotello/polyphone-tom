Le menu « Édition » contient les éléments suivants :

  • Annuler
    Annule la dernière action effectuée.
  • Rétablir
    Rétablit la dernière action annulée.
  • Copier
    Copie un élément, que ce soit dans l'arborescence ou dans un tableau.
  • Coller
    Colle un élément copié.
  • Supprimer
    Supprime un élément sélectionné, dans l'arborescence ou dans un tableau.
  • Renommer / Renommer en masse
    Renomme l'élément sélectionné. Des échantillons, instruments ou presets peuvent être renommés simultanément de plusieurs manières différentes :
    • écraser le nom existant avec le nom de la note en suffixe (échantillons uniquement),
    • écraser le nom existant avec un incrément en suffixe,
    • remplacement d'une chaîne de caractères en une autre,
    • insertion de texte à une position précise (permettant d'ajouter un préfixe ou un suffixe),
    • suppression d'une plage de caractères.
  • Préférences
    Ouvre les préférences du logiciel.
