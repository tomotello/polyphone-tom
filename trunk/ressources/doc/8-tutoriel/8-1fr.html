A partir de cet étape, il est considéré qu'un ou plusieurs échantillons sont disponibles pour la création d'un instrument.

Création d'un instrument et ajout d'échantillons

Tout d'abord, cliquer sur « Nouvel instrument » dans la barre d'outils si l'icône est présente, ou dans le menu contextuel de l'arborescence sur l'un des éléments de la soundfont en cours d'édition. Un nom est à renseigner pour notre nouvel instrument.

Ensuite, ajouter des échantillons dans l'instrument grâce à un « glisser / déposer » dans l'arborescence : des divisions apparaissent dans l'instrument.

Note : lorsque des divisions sont créées les échantillons ne sont pas copiés mais liés. Il n'y a aucune contrainte sur le nombre de divisions.

Disposition des divisions

Chaque division doit ensuite être disposée sur le clavier en modifiant le paramètre « Étendue note » dans le tableau de la page d'édition des instruments. Généralement, il est de bon usage de dimensionner l'étendue d'une division par rapport à la note de base de l'échantillon représenté. Toute la surface du clavier doit être couverte (par exemple de la note 36 à la note 96 pour un clavier classique de synthétiseur).

L'outil « Répartition automatique » permet de répartir automatiquement les divisions sur le clavier.

Paramétrage des divisions

Tout d'abord, si les échantillons sont bouclés la valeur « 1 » doit être renseignée dans la division globale pour le paramètre « Lecture en boucle ». La valeur « 3 » peut être utilisée si, en plus de la boucle, la fin de l'échantillon doit être lu après le relâchement de la note.

La division globale, comme son nom l'indique, permet de renseigner un paramètre de manière globale pour l'instrument. Un paramètre global ne s'applique que pour les divisions dont le même paramètre n'est pas renseigné - autrement dit les paramètres des divisions classiques sont prioritaires sur les paramètres de la division globale.

Dans le cadre d'un simple tutoriel aucun autre paramètre n'a besoin d'être édité pour que l'instrument fonctionne. Il est déjà possible de jouer à l'aide du clavier virtuel.

Cependant, pour améliorer l'instrument il est judicieux de renseigner une valeur dans « Vol env release (s) », ce qui évite au son de s'arrêter de manière abrupte dès que la touche est relachée. Les autres paramètres sont décrits dans la description des tableaux.
