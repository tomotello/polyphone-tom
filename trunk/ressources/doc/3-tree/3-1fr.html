Un clic droit sur un élément affiche un menu contenant les actions suivantes :
  • Nouvel échantillon...
    Permet l'import de nouveaux sons.
  • Nouvel instrument...
    Crée un nouvel instrument.
  • Nouveau preset...
    Crée un nouveau preset.
  • Associer à...
    Permet d'associer le ou les son(s) sélectionné(s) à un instrument, ou d'associer le ou les instrument(s) sélectionné(s) à un preset.
  • Remplacer par...
    Permet de remplacer un échantillon par un autre dans un instrument, ou de remplacer un instrument par un autre dans un preset. Les configurations de l'échantillon ou de l'instrument sont conservées.
  • Copier
    Copie le ou les élément(s) sélectionné(s).
  • Coller
    Colle le ou les élément(s) préalablement copié(s).
  • Supprimer
    Supprimer le ou les élément(s) sélectionné(s).
  • Renommer ou renommer en masse
    Renomme l'élément sélectionné. Des sons peuvent être renommés simultanément de plusieurs manières différentes :
    • écraser nom existant avec le nom de la note en suffixe,
    • écraser nom existant avec incrément en suffixe,
    • préfixe au nom existant,
    • suffixe au nom existant.
  • Fermer le fichier
    Ferme le fichier en cours d'édition.
