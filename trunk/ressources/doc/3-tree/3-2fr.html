L'opération « glisser / déposer » (ou copier / coller) dans l'arborescence permet d'effectuer rapidement des associations ou copies. Une sélection multiple est possible grâce aux boutons « ctrl » et « shift ».

Déplacement au sein de la même soundfont

  • Échantillon → instrument
    Association de l'échantillon à l'instrument, création d'une division dans l'instrument.
  • Instrument → preset
    Association de l'instrument au preset, création d'une division dans le preset.
  • Instrument → autre instrument
    Copie de toutes les divisions de l'instrument source dans l'instrument destinataire.
  • Division d'un instrument → autre instrument
    Copie de la division de l'instrument source dans l'instrument destinataire.
  • Preset → autre preset
    Copie de toutes les divisions du preset source dans le preset destinataire.
  • Division d'un preset → autre preset
    Copie de la division du preset source dans le preset destinataire.

Déplacement entre des soudfonts différentes

  • Échantillon
    Copie de l'échantillon.
  • Instrument
    Copie de l'instrument avec tous les échantillons associés.
  • Preset
    Copie du preset avec tous les instruments et échantillons associés.

Lors d'une copie d'éléments entre différentes soundfonts, si un élément du même nom existe déjà plusieurs options sont disponibles :

  • ignorer l'opération, dans quel cas la copie est annulée pour l'élément en question,
  • remplacer l'élément existant, dans quel cas l'élément existant est écrasé par l'élément copié,
  • dupliquer l'élément, dans quel cas l'élément est copié à côté de l'élément existant avec un nom différent.
