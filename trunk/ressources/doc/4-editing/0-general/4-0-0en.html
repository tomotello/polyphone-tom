The editing page of the general information allows us, in its upper part, to change the following information:

  • Name
    This is the internal name of the soundfont, which does not necessarily have to do with the file name. This name is used in the tree.
  • Author
    Relates to the author name of the soundfont.
  • Copyright
    Mention to fill if the file is subject to copyright.
  • Date
    Creation date of the file.
  • Product
    Product for which is designed the soundfont.
  • Comments
    Information not included in any of the preceding categories.
  • 24 bits samples
    Indicates whether the samples should be recorded in 24 bit (16 bits otherwise). Please note that 24-bit samples may not be supported in some synthesizers.
