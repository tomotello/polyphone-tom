Lors d'un clic sur un ou plusieurs instrument(s), la page d'édition des instruments apparaît.

Cette page contient plusieurs parties :

  • le tableau,
  • l'éditeur d'étendues lorsque le bouton « Éditer étendues » est cliqué,
  • la section modulateur (que l'on peut afficher ou masquer via le menu « Affichage »).

Si un seul instrument est sélectionné, la page proposera d'éditer les paramètres globaux de l'instrument ainsi que toutes ses divisions.

Dans le cas où plusieurs instruments sont sélectionnés, la page proposera d'éditer les paramètres globaux de chacun de ces instruments.
