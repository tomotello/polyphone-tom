The shortcut area contains some elements of the menu bar and of the contextual menu of the tree.

By default the shortcut area contains the following elements:

  • New file
    Create a new .sf2 file.
  • Open file
    Open an existing .sf2 file.
  • Save file
    Save the current file.
  • Undo
    Undo the last modification.
  • Redo
    Redo the last undone action.
  • New sample
    Add new samples to the current file.
  • New instruments
    Create a new instrument.
  • New preset
    Create a new preset.
  • Remove element
    Remove one or more elements selected in the tree or a table.
Changing the toolbar shortcuts is possible in the software settings.
